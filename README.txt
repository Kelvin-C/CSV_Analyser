/////////////// CSV ANALYSER ////////////////////////////

This software is used to read CSV files and plots graphs from the data.
This was developed specifically for csv files generated by Imperial College Union's eactivities.
The application is best used for analysing the finances of societies.

/////////////// FILES ////////////////////////////////////

- csv_analyser.py
	- Used to run the whole software, nothing else.

- gui.py
	- The code which contains all functions of the software. 

- menu.py
	- An attempt to create a modular design of the code. However, this was not completed.

/////////// SYSTEM REQUIREMENTS //////////////////////////

CSV Analyser had only been tested on the given system:

OS: Windows 10 64bit
RAM: 8 GB
CPU: Intel Core i7-4700MQ (2.40GHz, Quad Core)
